## Training Session Structure (90 minutes)

### Part 1: Framework Fundamentals (30 min)
1. **Two-Axis Model**: Affective vs Semantic preservation
   - How emotions/structure survive differently
   - H6 empirical coefficients: Poetry=68% affective, 8-25% semantic
   
2. **Survival Prediction Formula**: LM × A × S
   - Layer coefficient × Affective coefficient × Semantic coefficient
   - Aethelgard default: layers=2, affective=0.68, semantic=0.165 (T3)

3. **Preservation Quadrants & Interventions**
   - Quadrant 2 (High A/Low S): Add structural notes
   - Quadrant 3 (Low A/High S): Add affective hooks

### Part 2: Aethelgard Application (30 min)
1. **Aethelgard as T3 Creative Encoding**
   - Game mechanics as preservation strategy
   - Data fragments = encoded memories
   - Weather forecasts = atmospheric preservation
   
2. **Registry Integration for Aethelgard**
   - How to assess Aethelgard artifacts
   - Scoring game mechanics (affective/semantic)
   - Survival prediction for game states
   
3. **Practical Exercise**
   - Assess current Aethelgard state.json
   - Calculate survival prediction for game progress
   - Identify preservation opportunities

### Part 3: Tool Integration (30 min)
1. **Preservation Map Visualization**
   - View Aethelgard artifacts on coordinate plane
   - Track preservation quality over time
   - Identify at-risk game states
   
2. **T0 Seed Analysis**

   - Analyze seed overlap in Aethelgard actions

3. **Weekly Drift Monitoring**
